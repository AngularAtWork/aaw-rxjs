How To Create a Custom Pipeable Operator
How To:
  • Create a JavaScript function that takes no parameters or any number of required and/or optional parameters.
  • That function should return an unnamed Javascript function which takes an Observable as a parameter.
  • The unnamed function should return the same Observable passed in as a parameter with any custom pipeline processing.
When To Create Custom Pipeable Operators:
  • To make reusable pipeline code.
