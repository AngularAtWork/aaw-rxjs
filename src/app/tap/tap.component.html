tap
A pipeable, utility operator
What It Does:
  • It performs side effects while passing along the source Observable's value unchanged.
When To Use It:
  • While coding, use it to check the type of the value being passed from the source observable before it.
  • Anytime you want to perform a side-effect on the value from the source Observable or operator before it.

    For example, to console.log the value passed into it (for debugging or during development), or to emit a new value of a Subject.
