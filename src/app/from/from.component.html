from
A creation operator
What It Does:
  • Synchronously emits one or more elements of an array and then immediately completes.
  • Converts a Promise into an Observable.
  • Emits each individual character of a String.
When To Use It:
  • When you want to individually emit a given value or group of values.
  • When you want to individually process each item within an array return from an API call.
