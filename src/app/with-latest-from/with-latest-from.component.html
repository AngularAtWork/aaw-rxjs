withLatestFrom
A pipeable, join operator
What It Does:
  • Combines the source Observable with one or more inner Observables whose values are the last value emitted from each. The resulting emitted value is a tuple of the source and inner values. The source Observable and each of the inner Observables must have emitted at least once AND each of the inner Observables must emit before source Observable emits, in order that the function may emit its first value.
When To Use It:
  • To combine the values of Observables whose change in value you want to monitor, with the values of Observables whose emissions you don't want to monitor.
