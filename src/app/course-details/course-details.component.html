What is RxJS?
Reactive Extensions for JavaScript is a library that helps us code Observables
Using RxJS we can do all of the following:
  • Create Observables
  • Manipulate/transform the results of Observables
  • Create applications using the Reactive Programming Style. Using the Reactive Programming Style, we can avoid many of the challenges and issues associated with the Imperative Programming style
Who Is This Course For?
ANYONE who wants to learn how to effectively use RxJS
Do bear in mind:
  • The sample code and applications we'll be coding throughout the course will be done using Angular 14 and TypeScript.
  • We will discuss a few things that are specific to Angular programming only. However, that's a small portion of the course.  Most of the course focuses on RxJS as it can be used in any library or framework.
Which RxJS Features Will We Cover?
Subjects
Subject, BehaviorSubject, ReplaySubject, AsyncSubject
Creation/Combination Operators
combineLatest, defer, forkJoin, from, interval, merge, of, throwError, withLatestFrom, zip
Pipeable Operators
catchError, concatAll, concatMap, delay, debounceTime, distinctUntilChanged, exhaustMap, filter, finalize, groupBy, map, mergeAll, mergeMap, reduce, scan, share, shareReplay, switchMap, take, tap, throttleTime, toArray,
Build Custom Operators
log, sort, loading
What Will We Be Doing?
The objectives of this course:
  • Discuss in detail a number of RxJS features and what each of them does.
  • For each RxJS feature, we'll discuss some of the coding scenarios in which they each work most effectively.
  • As we learn a particular RxJS feature, we'll learn how to combine it with the other features we've learned to create reusable coding techniques.
  • Have a few side discussions which will help us in our effort to learn and better use RxJS.
  • Learn to parse through and understand the official RxJS documentation, including its marble diagrams.
