forkJoin
A creation, join operator
What It Does:
  • A function that accepts an array of Observables, emits an array consisting of the last value of each of the Observables and then completes. Each of the Observables in the array must complete in order that the function may emit its value.
When To Use It:
  • When you want your entire UI, that consists of multiple back end calls, to populate at once.
